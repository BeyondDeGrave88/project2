# Отчет о проведенной автоматизации

## Было запланировано:
- Создать проект на `Java11` с использованием `Selenide`, `JUnit5`.
- Настроить подключение к SQL-базе.
- Автоматизировать 17 сценариев оплаты тура.
- Подключить `Allure` для формирования отчетов.
- Написать документацию: `README.md, Plan.md, Report.md, Summary.md`.

## Было реализовано:
- SQL-запросы для проверки записей в БД, настроена автоматическая очистка БД.
- UI-тесты для сценариев с APPROVED и DECLINED картами.
- Подключен Allure.
- Оформлены баг-репорты в GitHub Issues.
- Написана вся необходимая документаци, включающая интсрукции по запуску проекта на других машинах и разных OS.

## Сработавшие риски:
- Отсутствие уникальных тестовых атрибутов - при изменении структуры страницы тесты могут не работать.
- Проблемы с логами - СУБД обрабатывает только два номера карт (APPROVED и DECLINED), остальные запросы приходилось обрабатывать вручную.
- Оценка времени.

## Общий итог по времени:
- Несмотря на возникшие риски, проект был завершен в запланированные сроки. Это стало возможным благодаря корректной первоначальной оценке сложности и резерву времени, заложенному на этапе планирования.
